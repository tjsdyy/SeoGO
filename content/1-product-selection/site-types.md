# 站点类型选择

> 选品的第一步是确定做什么类型的站点。不同类型的站点在 SEO 打法、变现模式和运营难度上差异巨大。本节介绍四种主流出海站点类型，帮你找到最适合自己的方向。

---

## 四种站点类型对比

| 类型 | 定义 | SEO 依赖度 | 变现模式 | 启动难度 | 适合人群 |
|------|------|-----------|----------|---------|---------|
| **游戏站** | 在线游戏平台、攻略站、游戏工具 | 极高 | IAA 广告 + IAP | 中 | 有游戏领域经验 |
| **AI 工具站** | 基于 AI API 的垂直工具产品 | 高 | 订阅/Credit制 | 低-中 | 有技术背景 |
| **导航站** | 精选推荐某领域优质资源 | 高 | 广告 + 付费收录 | 低 | 个人站长 |
| **铲子站** | 为行业参与者提供效率工具/SaaS | 高 | SaaS 订阅 | 中-高 | 有行业洞察 |

---

## 游戏站

游戏和泛娱乐是中国出海最成熟的赛道之一。SEO 在以下游戏相关场景中价值极高：

1. **在线网页游戏平台**：流量几乎 100% 来自搜索（CrazyGames、Poki）
2. **游戏攻略/Wiki 站**：纯 SEO 驱动的内容赛道
3. **游戏工具站**：计算器、模拟器、数据库等天然搜索需求
4. **短剧/网文平台**：新兴赛道，ReelShort 已验证模式

### 主要品类与 SEO 机会

| 品类 | 代表产品 | SEO 依赖度 | 变现模式 |
|------|----------|-----------|----------|
| 在线小游戏平台 | CrazyGames、Poki | 极高 | IAA |
| 工具类游戏网站 | 攻略站、计算器 | 高 | 广告 + 联盟 |
| 休闲/超休闲游戏 | 各类小游戏 | 中 | IAA + IAP |
| 短剧/网文 | ReelShort、Dreame | 中 | IAP + 订阅 |

### 关键词结构示例

```
核心词：online games, free games
品类词：racing games, puzzle games, shooting games
特征词：2 player games, unblocked games
长尾词：free online racing games for kids no download
```

### 页面结构设计

```
首页 → 覆盖 "free online games"
├── /category/racing/ → "free racing games online"
├── /category/puzzle/ → "free puzzle games online"
│   ├── /game/[游戏名]/ → 具体游戏页面
│   └── /tags/2-player/ → "2 player puzzle games"
├── /new/ → "new online games"
└── /popular/ → "popular online games"
```

### SEO 特殊挑战

- **时效性强**：新版本上线时搜索量暴涨，过后迅速下降 → 建立内容更新 SOP
- **版权合规**：使用游戏截图可能涉及版权 → 了解厂商的创作者政策
- **国际化**：不同地区玩家偏好不同 → 根据目标市场选对应品类

### 变现模式

| 广告类型 | 位置 | 预期 eCPM（美区） |
|----------|------|-------------------|
| Display Banner | 侧边栏/页面底部 | $1-5 |
| 插屏广告 | 游戏切换时 | $5-15 |
| 激励视频 | 游戏内奖励 | $10-30 |
| Native Ad | 内容流中 | $3-8 |

## AI 工具站

AI 浪潮为出海创业者带来了前所未有的机会窗口。对大多数出海团队来说，机会主要在**应用层**和**垂直层**。

### AI 产品主要品类

| 品类 | 代表产品 | SEO 机会 | 竞争强度 |
|------|----------|----------|----------|
| AI 写作 | Jasper、Copy.ai | 中（已红海） | 极高 |
| AI 视频 | Runway、Pika、HeyGen | 高 | 中-高 |
| AI 音频/音乐 | ElevenLabs、Suno | 高 | 中 |
| AI 办公 | Gamma、Beautiful.ai | 高 | 中 |
| AI 教育 | Duolingo AI、Photomath | 高 | 中 |
| AI SEO 工具 | Surfer SEO、NeuronWriter | 高 | 中 |
| AI + 垂直行业 | 法律/房产/电商 AI | 极高 | 低-中 |

### 三种出海路径

| 路径 | 特点 | 启动速度 | 壁垒 |
|------|------|---------|------|
| **API 套壳** | 基于 OpenAI 等 API 包装 | 1-4 周 | 低 |
| **AI-Native** | 自研模型，产品即 AI | 3-12 月 | 高 |
| **AI + 垂直** | AI 能力应用到细分行业 | 2-6 月 | 中-高 |

> **推荐路径**：AI + 垂直行业。大公司不会优先覆盖细分垂直场景，行业知识本身就是壁垒。

### 高价值关键词模式

```
功能型："AI [功能] [tool/generator/maker]"
  例：AI video generator, AI logo maker

场景型："AI [场景/行业] [tool/assistant]"
  例：AI real estate copywriter, AI resume builder

替代型："[AI产品] alternatives / free"
  例：ChatGPT alternatives, free Midjourney alternative

教程型："how to use [AI产品/技术]"
  例：how to use Stable Diffusion
```

### 内容策略

1. **Prompt 库**：用 Prompt 模板吸引搜索流量
2. **AI 工具目录**："Best AI tools for [场景]" 系列
3. **教程与用例**：深度使用教程和实际案例

## 导航站

导航站（Navigation/Directory Site）是出海 SEO 的经典模式，通过整合信息、组织资源获取搜索流量。

### 导航站的天然 SEO 优势

- **海量页面**：每条收录的资源都是独立页面，覆盖大量长尾词
- **结构化数据**：评分、分类、价格信息适合 Schema Markup
- **UGC 内容**：评论和评价持续为页面增加新鲜内容
- **内链密度高**：分类页、标签页、对比页形成密集内链网络

### 页面类型规划

| 页面类型 | 目标关键词 | 示例 |
|----------|------------|------|
| 首页 | 核心品类词 | "AI writing tools" |
| 分类页 | 品类 + 细分 | "AI writing tools for marketing" |
| 详情页 | 品牌词 + review | "Jasper AI review" |
| 对比页 | A vs B | "Jasper vs Copy.ai" |
| Best-of 列表 | best + 品类 | "10 best AI writing tools" |
| 替代品页 | alternatives to X | "Jasper alternatives" |

### 选择细分领域

```
✅ 适合做导航站的领域：
- 产品/工具数量多（>50 个）
- 存在明确的对比需求
- 现有聚合平台不够好（信息不全、更新慢）

❌ 不适合的领域：
- 产品数量太少
- 已有权威聚合平台（如 G2 对 SaaS）
- 信息变化极快，维护成本太高
```

### 变现路径

| 变现方式 | 适用阶段 | 预期 RPM | 门槛 |
|----------|----------|---------|------|
| Google AdSense | 初期 | $2-8 | 基本无 |
| Mediavine | 中期 | $15-30 | 月 50K Sessions |
| 联盟营销 | 全阶段 | 因产品而异 | 需申请联盟计划 |
| 付费收录 | 中后期 | 固定收费 | 需要一定流量 |

## 铲子站（工具类 + SaaS）

> "淘金热中最赚钱的不是淘金者，而是卖铲子的人。"

当某个行业蓬勃发展时，为行业参与者提供效率工具，往往比直接参与竞争更稳健。

### 卖铲子的经典案例

| 淘金热（行业） | 卖铲子（工具） |
|---------------|---------------|
| 电商卖家爆发 | Shopify（建站）、Oberlo（选品） |
| 内容创作者爆发 | Canva（设计）、Descript（视频编辑） |
| AI 应用爆发 | Langchain（框架）、Pinecone（向量数据库） |
| 独立站 SEO 需求 | Ahrefs（SEO 工具）、Surfer SEO |

### 如何选方向

**方法 1：痛点挖掘法** — 从用户工作流中找痛点

```
用户工作流程：
写博客 → [找关键词] → [写大纲] → [写正文] → [配图] → [发布] → [推广]

每个环节都是一个工具机会
```

**方法 2：搜索需求验证法** — 用搜索数据验证需求

```
"[动作] + tool"        → "keyword research tool"
"[动作] + online"      → "compress PDF online"
"free [功能] + tool"   → "free grammar checker"
"[功能] + generator"   → "invoice generator"
```

**方法 3：竞品缺陷法** — 找到现有工具的不足

- 太贵 → 做价格低 50-80% 的替代品
- 太复杂 → 做更简单的单一场景版本
- 不支持某场景 → 针对被忽视的场景适配

### 免费工具矩阵策略

用免费工具获取流量，用增值功能转化付费。以 Ahrefs 为例：

```
├── Free Backlink Checker      → "backlink checker"
├── Free Keyword Generator     → "keyword generator"
├── Free SERP Checker          → "serp checker"
├── Free Website Authority     → "domain authority checker"
└── Free Broken Link Checker   → "broken link checker"
```

### SaaS 增长飞轮

```
免费工具/内容 → 获取搜索流量
    ↓
用户注册 → 体验产品价值
    ↓
付费转化 → 产生收入
    ↓
用户口碑 → 产生外链和品牌搜索
    ↓
SEO 权重提升 → 更多搜索流量（正向循环）
```

## 如何选择站点类型

### 决策矩阵

| 考量因素 | 游戏站 | AI 工具站 | 导航站 | 铲子站 |
|---------|--------|----------|--------|--------|
| 技术门槛 | 中 | 中 | 低 | 中-高 |
| 资金需求 | 低-中 | 低-中 | 低 | 中-高 |
| 内容需求 | 高 | 中 | 高 | 中 |
| 变现速度 | 中 | 快 | 慢 | 中 |
| 长期价值 | 中 | 中 | 中 | 高 |
| 单人可操作 | 是 | 是 | 是 | 困难 |

### 推荐选择路径

```
★★★★★ AI + 垂直行业工具 → 市场空间大，壁垒可建立
★★★★☆ 导航站/聚合站 → 启动快，适合个人站长
★★★★☆ 在线游戏平台 → 纯 SEO 驱动，流量天花板高
★★★☆☆ 游戏攻略/工具站 → 需要领域深耕
★★★☆☆ AI 套壳产品 → 快速验证，但需尽快建立差异化
```
