# 关键词分析

> 关键词分析是网站架构的第一步。选择错误的关键词，后面的所有工作都是在浪费资源。本节将系统讲解如何评估搜索量大小、解读关键词难度（KD）、利用 KGR 黄金比率筛选低竞争机会词、分类搜索意图，并最终搭建一套可落地执行的关键词优先级矩阵。

---

## 一、流量大小评估

### 1.1 搜索量的含义

月搜索量（Monthly Search Volume）是指一个关键词在某个国家/地区每月被搜索的平均次数。它是评估关键词流量潜力最直接的指标。

**常用数据来源**：

| 工具 | 数据来源 | 特点 |
|------|---------|------|
| Google Keyword Planner | Google 官方数据 | 免费，但给出的是范围值（如 1K-10K） |
| Ahrefs Keywords Explorer | 点击流数据 + Google 数据 | 精确数值，支持 200+ 国家 |
| SEMrush Keyword Magic Tool | 自有数据库 | 支持关键词分组和过滤 |
| Ubersuggest | Google 数据 | 免费额度有限，适合新手 |

### 1.2 搜索量的正确解读

搜索量不等于你能获得的流量。你还需要考虑：

- **CTR 分布**：排名第 1 的页面平均 CTR 约 27%，第 10 名约 2.5%
- **SERP 特征挤压**：精选摘要、广告、知识面板会大量分走点击
- **零点击搜索**：约 50% 的搜索不会产生任何点击（用户直接在 SERP 获取答案）

**预估流量公式**：

```
预估月流量 = 月搜索量 × 预期排名位置的 CTR × (1 - 零点击率)

示例：
搜索量 5,000 × 排名第 3 的 CTR 10% × (1 - 30%) = 350 次/月
```

### 1.3 搜索量分级参考

| 搜索量范围 | 级别 | 适合的网站阶段 |
|-----------|------|--------------|
| 10,000+ | 头部词（Head Terms） | 高权重成熟站点 |
| 1,000-10,000 | 中部词（Body Terms） | 有一定权重的成长期站点 |
| 100-1,000 | 长尾词（Long-Tail） | 新站首选目标 |
| 10-100 | 微长尾词 | 精准转化或 programmatic SEO |

> **建议**：新站优先瞄准月搜索量 100-1,000 的长尾词。这些词竞争较低，更容易快速获得排名和流量信心。

## 二、关键词难度 KD

### 2.1 什么是关键词难度

关键词难度（Keyword Difficulty / KD）是衡量一个关键词在自然搜索中获得排名的难度。不同工具的计算方式不同，但核心逻辑类似：**评估当前排名靠前的页面有多强**。

### 2.2 各工具的 KD 评分对比

| 工具 | 评分范围 | 主要依据 | 特点 |
|------|---------|---------|------|
| Ahrefs KD | 0-100 | 排名前 10 页面的 Referring Domains 数量 | 偏重外链因素 |
| SEMrush KD% | 0-100 | 排名页面的权威度、外链、内容质量 | 综合因素 |
| Moz KD | 0-100 | Page Authority 和 Domain Authority | 偏重域名权威 |
| KWFinder KD | 0-100 | 多维度综合评分 | 对新手友好 |

> **重要提醒**：不同工具的 KD 评分不可直接对比。Ahrefs 的 KD 30 和 SEMrush 的 KD 30 含义不同。建议在同一个项目中统一使用一个工具的 KD 数据。

### 2.3 KD 实操解读（以 Ahrefs 为例）

| KD 范围 | 难度等级 | 含义 | 新站是否可做 |
|---------|---------|------|------------|
| 0-10 | 极低 | 几乎不需要外链即可排名 | 优先目标 |
| 11-30 | 低 | 需要少量外链和优质内容 | 推荐目标 |
| 31-50 | 中等 | 需要较多外链和强内容 | 中期目标 |
| 51-70 | 高 | 需要大量高质量外链 | 长期规划 |
| 71-100 | 极高/近乎不可能 | 被超级权威站点垄断 | 暂时放弃 |

### 2.4 KD 评分的局限性

KD 分数不是万能的，需要注意以下盲区：

- **只看外链数量，忽略内容质量**：有些低 KD 的词，排名前列的内容质量极高
- **未考虑 SERP 特殊特征**：精选摘要、Knowledge Panel 会挤压有机排名的点击率
- **未考虑品牌效应**：某些词被大品牌垄断，即使 KD 不高也难以竞争
- **数据延迟**：新出现的竞争页面可能尚未被工具收录

### 2.5 手动 SERP 分析补充 KD

除了工具 KD 评分，你还需要手动分析 SERP 来评估真实难度：

```
SERP 分析检查项：

□ 排名前 10 的网站 DR/DA 是多少？
  → 如果全是 DR 80+ 的网站，竞争极其激烈

□ 排名前 10 的页面外链数量（Referring Domains）？
  → 与你现有页面的外链水平对比

□ 排名前 10 的内容质量如何？
  → 内容过时/质量差 = 机会；内容优秀且持续更新 = 高难度

□ 是否有新站/低权重站点在排名？
  → 如果有，说明 Google 认为现有结果不够好，存在机会

□ 排名前 10 的页面最后更新时间？
  → 内容老旧的 SERP 更容易突破
```

## 三、KGR 黄金比率

### 3.1 什么是 KGR

KGR（Keyword Golden Ratio）是由 Doug Cunnington 提出的一种快速筛选低竞争关键词的方法。其公式为：

```
KGR = Google "allintitle" 搜索结果数 / 月搜索量

条件：月搜索量 < 250
```

**allintitle 搜索**：在 Google 搜索栏输入 `allintitle:关键词`，返回的结果数表示有多少页面在 Title 中完整包含了该关键词。

### 3.2 KGR 数值解读

| KGR 数值 | 含义 | 建议 |
|----------|------|------|
| < 0.25 | 黄金机会 | 发布后大概率几天内进入前 50 |
| 0.25 - 1.0 | 可行机会 | 发布后有机会在数周内获得排名 |
| > 1.0 | 竞争较大 | 即使搜索量低，也不容易快速排名 |

### 3.3 KGR 实操步骤

**第 1 步：筛选候选关键词**

在关键词工具中筛选月搜索量在 50-250 之间的长尾关键词。

**第 2 步：获取 allintitle 结果数**

在 Google 中搜索 `allintitle:你的关键词`，记录返回的结果数量。

**第 3 步：计算 KGR**

```
示例：
关键词：best budget tripod for vlogging
月搜索量：150
allintitle 结果数：18

KGR = 18 / 150 = 0.12 → 黄金机会！
```

**第 4 步：批量计算**

建议在电子表格中批量计算，快速筛选出所有 KGR < 0.25 的关键词。

### 3.4 KGR 的适用场景与局限

**适用场景**：
- 新站冷启动，需要快速获得排名
- 长尾词批量挖掘
- Programmatic SEO 的关键词验证

**局限性**：
- 只适用于搜索量 < 250 的词，不适用于头部词
- allintitle 数据不是实时的，可能有偏差
- 不考虑内容质量和外链因素

## 四、搜索意图分类

理解搜索意图是关键词分析的核心。Google 在其《搜索质量评估指南》中将搜索意图分为四类：

### 4.1 四种搜索意图

| 意图类型 | 用户目的 | 典型修饰词 | 占比 | 最佳内容类型 |
|---------|---------|-----------|------|------------|
| **信息型** | 了解知识、获取信息 | what, how, why, guide, tutorial | 60-70% | 博客、教程、深度指南 |
| **导航型** | 前往特定网站/页面 | 品牌名, login, official site | 10-15% | 品牌页面、登录页 |
| **商业调查型** | 研究对比，准备购买 | best, vs, review, top, alternative | 10-15% | 对比文章、评测、排行榜 |
| **交易型** | 完成操作（购买/注册/下载） | buy, price, discount, download, free trial | 10-15% | 产品页、落地页、定价页 |

### 4.2 如何判断搜索意图

**方法一：SERP 分析法（最准确）**

搜索关键词后，观察排名前 10 的结果类型：

| SERP 特征 | 对应意图 |
|-----------|---------|
| 知识面板（Knowledge Panel） | 信息型 |
| 精选摘要（Featured Snippet） | 信息型 |
| People Also Ask | 信息型 |
| 购物广告（Shopping Ads） | 交易型 |
| 评测类文章占主导 | 商业调查型 |
| 品牌官网排第一 | 导航型 |

**方法二：修饰词判断法（快速）**

通过关键词中的修饰词快速判断意图：

```
信息型修饰词：
  what, how, why, when, guide, tutorial, tips, examples, definition

商业调查修饰词：
  best, top, review, comparison, vs, alternative, recommended

交易型修饰词：
  buy, purchase, price, cost, cheap, discount, coupon, download, free trial

导航型修饰词：
  [品牌名], login, sign in, official site, app, dashboard
```

### 4.3 一个页面对应一个主要意图

> **核心原则**：每个页面应针对一种主要搜索意图进行优化。混合意图的页面往往哪个意图都排不好。

**正确做法**：为不同意图分别创建独立页面，通过内链相互引导，形成从信息型到商业调查到交易型的转化路径。

## 五、关键词优先级矩阵

### 5.1 双维度矩阵模型

以**价值（Value）**和**难度（Difficulty）**为两个轴，将关键词分为四个象限：

```
高价值 ↑
         │  ★ Quick Wins     │  ⚡ 战略关键词
         │  低难度 + 高价值    │  高难度 + 高价值
         │  → 立即执行         │  → 长期规划
         │                   │
    ─────┼───────────────────┼─────── → 高难度
         │                   │
         │  ✗ 可选/延后       │  ⚠ 不建议投入
         │  低难度 + 低价值    │  高难度 + 低价值
         │  → 有余力时做      │  → 放弃
低价值   │                   │
```

### 5.2 价值评分标准

| 评分维度 | 权重 | 评分说明 |
|---------|------|---------|
| 月搜索量 | 20% | 直接决定流量天花板 |
| 业务相关度 | 30% | 与你的产品/服务的匹配程度 |
| 转化潜力 | 25% | 搜索意图是否接近转化环节 |
| 流量价值（CPC） | 15% | 广告主愿意出价越高，商业价值越大 |
| 趋势方向 | 10% | 上升趋势加分，下降趋势减分 |

### 5.3 难度评分标准

| 评分维度 | 权重 | 评分说明 |
|---------|------|---------|
| 工具 KD 评分 | 40% | Ahrefs 或 SEMrush 的 KD 数值 |
| 排名竞争者 DR | 25% | 前 10 名网站的平均 DR |
| 内容质量差距 | 20% | 现有排名内容的质量与你的差距 |
| 所需外链投入 | 15% | 预估需要获取的外链数量 |

### 5.4 四象限执行策略

| 象限 | 特征 | 策略 | 时间线 |
|------|------|------|--------|
| **Quick Wins** | 高价值 + 低难度 | 立即创建/优化内容 | 1-4 周 |
| **战略关键词** | 高价值 + 高难度 | 长期内容建设 + 外链获取 | 3-12 个月 |
| **可选/延后** | 低价值 + 低难度 | 有余力时做，或批量处理 | 视资源而定 |
| **不建议投入** | 低价值 + 高难度 | 除非有特殊战略意义，否则放弃 | 不投入 |

### 5.5 Quick Wins 的五种来源

1. **低 KD + 中搜索量**：KD < 20，月搜索量 200-2000
2. **已有排名 11-20 位的关键词**：在 Search Console 中筛选 Position 11-20
3. **竞品排名但内容质量差**：通过 SERP 手动分析发现内容过时的机会
4. **新兴关键词（Rising Keywords）**：Google Trends 中的 Breakout 标签
5. **FAQ / People Also Ask 关键词**：问答型关键词通常 KD 较低

## 六、实战：搭建关键词优先级表格

### 6.1 推荐表格结构

```
| 关键词 | 月搜索量 | KD | KGR | CPC | 搜索意图 | 业务相关度(1-10) |
| 转化潜力(1-10) | 价值总分 | 难度总分 | 象限分类 | 对应页面URL | 当前排名 | 状态 |
```

### 6.2 操作步骤

1. **导出数据**：从 Ahrefs/SEMrush 导出关键词数据（搜索量、KD、CPC）
2. **计算 KGR**：对搜索量 < 250 的词批量计算 KGR
3. **标注意图**：为每个关键词标注搜索意图类型（I/N/C/T）
4. **人工评分**：对业务相关度和转化潜力进行人工评分（1-10）
5. **计算总分**：使用加权公式分别计算价值总分和难度总分
6. **矩阵分类**：将关键词分入四个象限
7. **排序执行**：在每个象限内按优先级排序
8. **指定页面**：为每个关键词指定对应的内容页面 URL
9. **设置状态**：待创建 / 进行中 / 已发布 / 已排名

### 6.3 定期复盘

建议每月更新一次优先级矩阵：

- 检查已执行关键词的排名变化
- 更新搜索量和 KD 数据
- 调整关键词的优先级分类
- 补充新发现的关键词（包括 KGR 筛选）
- 移除已放弃或已完成的关键词
