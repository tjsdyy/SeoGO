> 工欲善其事，必先利其器。在正式开始海外 SEO 之旅前，你需要完成一系列基础设施搭建和技能储备。本章将帮助你从零开始，系统性地准备好所有必要的账号、工具、资金通道和基础知识。

## 本章导读

准备阶段分为两大板块：**基础设施搭建**和**必备技能储备**。前者确保你拥有开展海外业务所需的全部账号、工具和服务；后者确保你具备独立操作和持续学习的能力。

建议按顺序阅读，因为后面的章节会依赖前面的准备工作。例如，注册 Google 账号体系前你需要先准备好 VPN 工具，而配置云服务和 CDN 前你需要先完成域名注册。

## 学习建议

1. **预计时间**：完成全部准备工作大约需要 1-2 周
2. **优先级**：如果时间紧张，先完成 1.1.1-1.1.6 和 1.2.2，其他内容可以边做边学
3. **动手实践**：每个章节都包含实操步骤，建议边读边做，不要只看不练
4. **记录问题**：准备一个笔记文档，记录过程中遇到的问题和解决方案

## 完成标准

当你完成本章所有内容后，你应该具备：

- [x] 一张可用于海外支付的银行卡
- [x] 至少一个收款渠道（Stripe 或 PayPal）
- [x] 稳定的网络环境和必备工具
- [x] 一个已注册的域名
- [x] 完整的 Google 账号体系
- [x] 基本的 HTML/CSS 阅读能力
- [x] 对 SEO 核心概念的理解
- [x] 至少一个 AI 工具的熟练使用能力
