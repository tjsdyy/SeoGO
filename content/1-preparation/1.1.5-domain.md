> 域名是你在互联网上的地址，也是品牌形象的重要组成部分。选择一个好域名并正确注册，是建站之前最重要的决策之一。本节将介绍域名选择的核心策略、注册流程以及常见的坑。

## 域名基础知识

### 域名的组成

```
https://www.example.com/blog/article
  |      |      |      |    |     |
协议   子域名  二级域名 TLD  路径   页面
```

- **顶级域名（TLD）**：.com、.net、.org、.io 等
- **二级域名**：你注册的主体部分，如 example
- **子域名**：www、blog、shop 等，可自行创建

### 域名对 SEO 的影响

域名本身对 SEO 的直接影响已经很小，但间接影响不可忽视：

1. **品牌辨识度**：好记的域名带来更多直接访问和品牌搜索
2. **点击率**：搜索结果中，简短清晰的域名更容易获得点击
3. **外链获取**：有品牌感的域名更容易获得其他网站的引用
4. **用户信任**：.com 域名的用户信任度普遍高于其他 TLD

## 域名选择策略

### 核心原则

1. **简短**：越短越好，控制在 15 个字符以内
2. **好记**：容易拼写、容易发音
3. **避免歧义**：不会被误拼或误解
4. **品牌化**：可以作为品牌名使用，而非纯关键词堆砌
5. **无侵权**：不使用已注册的商标

### TLD（顶级域名）选择

| TLD | 特点 | 适用场景 | 推荐度 |
|-----|------|----------|--------|
| **.com** | 全球认知度最高 | 通用商业网站 | 强烈推荐 |
| **.net** | 技术和网络相关 | 工具类、技术类网站 | 推荐 |
| **.org** | 组织和非营利 | 社区、开源项目 | 一般 |
| **.io** | 技术和创业圈流行 | SaaS、开发者工具 | 推荐 |
| **.co** | 简短的 .com 替代 | 创业公司 | 一般 |
| **.ai** | AI 相关项目 | AI 工具和服务 | 推荐（AI 领域） |
| **.dev** | Google 运营，强制 HTTPS | 开发者工具 | 推荐 |
| **ccTLD** | 国家代码域名（.us/.uk/.de） | 针对特定国家市场 | 特定场景推荐 |

> **建议**：如果能拿到 .com，优先选择 .com。如果做技术/SaaS 类产品，.io 和 .dev 也是好选择。避免使用冷门 TLD，如 .xyz、.club 等，用户信任度较低。

### 命名策略

#### 方法一：品牌词（最推荐）

创造一个独特的品牌词，如：
- Ahrefs（无实际含义，但好记）
- Canva（简短、易发音）
- Notion（有含义但独特）

**优点**：独特性强、商标保护容易、不受关键词限制

#### 方法二：组合词

将两个有意义的短词组合，如：
- MailChimp = Mail + Chimp
- WordPress = Word + Press
- YouTube = You + Tube

**优点**：有含义、好记、通常有 .com 可用

#### 方法三：关键词 + 修饰词

在行业关键词基础上加修饰词，如：
- SmartSEO
- SEObuddy
- RankTracker

**优点**：直观表达业务、新访客容易理解

**缺点**：品牌感较弱、域名可能已被注册

#### 应避免的命名方式

- 纯关键词域名（如 best-seo-tools.com）——Google 已降低这类域名的权重
- 包含连字符的域名（如 my-brand-name.com）——看起来不专业
- 使用数字替代字母（如 gr8tools.com）——容易混淆
- 过长的域名——难以记忆和输入
- 与知名品牌相似的域名——可能涉及商标侵权

### 推荐方案

**首选：Cloudflare Registrar**
- 以成本价（批发价）注册域名，没有加价
- 与 Cloudflare CDN 和 DNS 服务完美集成
- 免费的 WHOIS 隐私保护
- 后续可以直接在 Cloudflare 上配置 DNS、SSL、CDN

**备选：Namecheap**
- 界面友好，新手容易上手
- 首年价格便宜
- 免费 WHOIS 隐私保护
- 支持更多 TLD 选择

## 域名注册实操

### Cloudflare 注册流程

1. 注册 Cloudflare 账号（cloudflare.com）
2. 进入 Dashboard，点击左侧 "Registrar"
3. 搜索你想要的域名
4. 选择 TLD 和注册年限（建议先注册 1 年）
5. 填写注册信息（WHOIS 信息会自动隐私保护）
6. 完成支付（支持信用卡、PayPal）

### 注册后必做设置

1. **开启 WHOIS 隐私保护**
   - 隐藏你的个人姓名、地址、电话和邮箱
   - Cloudflare 和 Namecheap 默认开启

2. **配置 DNS 记录**
   ```
   类型    名称    值                    TTL
   A       @       你的服务器 IP          Auto
   CNAME   www     你的域名.com           Auto
   MX      @       邮件服务器地址         Auto
   TXT     @       各类验证记录           Auto
   ```

3. **启用 DNSSEC**
   - 防止 DNS 劫持和缓存投毒
   - Cloudflare 一键开启

4. **开启域名锁定（Transfer Lock）**
   - 防止域名被恶意转移
   - 需要转移时再手动关闭

5. **设置自动续费**
   - 避免域名到期后被他人抢注
   - 确保绑定的支付方式有效

## 域名购买与投资建议

### 关于购买二手域名

有时你想要的域名已被注册，可以考虑购买：

- **域名交易平台**：Dan.com、Sedo、Afternic、Flippa
- **直接联系持有者**：通过 WHOIS 查询找到持有者联系方式
- **拍卖过期域名**：在 NameJet、SnapNames 上竞拍

### 购买二手域名的注意事项

1. **检查域名历史**：使用 Wayback Machine（web.archive.org）查看域名过去的内容
2. **检查外链情况**：用 Ahrefs 查看域名是否有垃圾外链
3. **检查是否被惩罚**：在 Google 中搜索 site:域名，看是否有索引
4. **避免有不良历史的域名**：赌博、色情、诈骗相关的域名，即使过期了也可能被 Google 标记

### 品牌保护建议

当你的业务开始增长后，考虑：

- 注册域名的多个 TLD（.com + .net + .org）
- 注册常见的拼写错误域名
- 注册对应的社交媒体用户名（Twitter/X、Facebook、Instagram）
- 考虑注册商标（尤其在目标市场国家）

## 域名检查清单

在最终注册域名前，完成以下检查：

- [ ] 域名简短、好记、好拼写
- [ ] 没有商标冲突（在 USPTO、EUIPO 等商标局搜索）
- [ ] .com 版本可用（或你有充分理由选择其他 TLD）
- [ ] 在 Google 中搜索域名，没有负面关联
- [ ] 使用 Wayback Machine 检查没有不良历史
- [ ] 发音清晰，口头传播时不会被误解
- [ ] 社交媒体上相关用户名可用
- [ ] 注册商选择合适，已了解续费价格
- [ ] 已开启 WHOIS 隐私保护
- [ ] 已设置自动续费
