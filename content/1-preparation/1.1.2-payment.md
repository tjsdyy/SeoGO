> 当你的海外网站开始产生收入时，你需要一个可靠的收款渠道将资金收回。本节将深入对比三大主流收款平台——Stripe、PayPal 和 Wise，帮助你根据业务类型选择最合适的方案。

## 收款渠道选择的核心考量

在选择收款渠道时，你需要关注以下几个维度：

- **支持的支付方式**：信用卡、借记卡、本地支付等
- **费率结构**：交易手续费、提现费、货币转换费
- **到账速度**：从客户付款到你收到资金的时间
- **支持的国家和地区**：你的目标客户所在地
- **集成难度**：接入你网站的技术门槛
- **风控政策**：资金被冻结或扣留的风险

## Stripe

### 概述

Stripe 是全球领先的在线支付处理平台，被 Shopify、Amazon、Google 等巨头采用。它以开发者友好和强大的 API 著称，是做海外独立站收款的首选。

### 费率结构

| 费用类型 | 标准费率 | 说明 |
|----------|----------|------|
| 标准交易费 | 2.9% + $0.30 | 每笔交易 |
| 国际卡交易 | +1% | 非本国发行的卡 |
| 货币转换 | +1% | 需要转换货币时 |
| 退款手续费 | $15 | 每笔退款（部分地区免除） |
| 提现费 | 免费 | 到达关联银行账户 |
| 争议处理费 | $15 | 客户发起 chargeback |

### 注册要求

Stripe 目前不直接支持中国大陆企业注册，你需要通过以下方式之一：

1. **香港公司 + 香港银行账户**（推荐）
   - 注册香港公司（可通过代理，费用约 $500-1000）
   - 使用香港银行账户作为收款账户
   - 在 Stripe 香港站注册

2. **美国公司 + 美国银行账户**
   - 通过 Firstbase、Stripe Atlas 等服务注册美国 LLC
   - 开设美国银行账户（如 Mercury Bank）
   - 在 Stripe 美国站注册

3. **使用 Stripe 第三方集成商**
   - LemonSqueezy、Paddle 等平台底层使用 Stripe
   - 你作为卖家入驻，平台处理支付合规
   - 费率略高，但省去了公司注册的麻烦

### Stripe 接入方式

```
Stripe 提供多种接入方式：

1. Stripe Checkout（最简单）
   - 托管支付页面，无需自建支付表单
   - 适合快速上线的独立站

2. Stripe Elements（推荐）
   - 可嵌入的支付组件
   - 可自定义样式，融入你的网站设计

3. Stripe API（高级）
   - 完全自定义的支付流程
   - 适合有技术能力的团队
```

### Stripe 优势与劣势

**优势：**
- 支付成功率高，智能路由优化
- 支持 135+ 种货币
- 强大的 Dashboard 和数据分析
- 支持订阅、一次性付款、发票等多种模式
- 欺诈防护系统 Stripe Radar

**劣势：**
- 不支持中国大陆直接注册
- 需要有一定的技术能力进行集成
- 对高风险行业审核严格

## PayPal

### 概述

PayPal 是全球最知名的在线支付品牌，拥有超过 4 亿活跃用户。对于刚起步的个人创业者来说，PayPal 可能是最容易上手的收款方式。

### 费率结构

| 费用类型 | 标准费率 | 说明 |
|----------|----------|------|
| 标准交易费 | 3.49% + $0.49 | 商业收款 |
| 国际交易 | +1.5% | 跨境交易附加费 |
| 货币转换 | 3%-4% | PayPal 汇率加点较高 |
| 提现到银行 | 免费（美国） | 其他地区可能收费 |
| 提现到港卡 | HK$0-3.5 | 视金额和速度 |
| 退款手续费 | 不退还手续费 | 原交易手续费不退 |

### 账户类型

1. **个人账户**：只能用于付款，不能收取商业款项
2. **商业账户**（推荐）：可以收款、发送发票、集成到网站

### 注册流程

1. 访问 PayPal 对应地区官网
2. 选择"商业账户"注册
3. 填写商业信息（个人 freelancer 也可选择个体经营）
4. 验证邮箱和手机号
5. 关联银行账户或银行卡
6. 完成身份验证

### PayPal 接入方式

- **PayPal Buttons**：最简单，复制粘贴按钮代码到网页
- **PayPal Checkout**：更好的用户体验，支持信用卡直付
- **PayPal Invoicing**：发送电子发票，适合 B2B 和自由职业

### PayPal 优势与劣势

**优势：**
- 全球知名度高，客户信任度高
- 注册简单，个人即可开通商业账户
- 买家保护政策完善
- 无需技术能力即可使用（发票、支付链接）

**劣势：**
- 费率偏高，尤其是跨境交易
- 货币转换汇率差
- 资金冻结风险较高（尤其是新账户）
- 争议处理偏向买家
- 账户关联风险（多账户可能被封）

> **警告**：PayPal 对新账户的风控非常严格。新注册的商业账户在前几个月内，大额收款很可能被临时冻结 21 天。建议前期从小额交易开始，逐步积累信用。

## Wise Business

### 概述

Wise Business 不是传统的支付网关，而是一个多币种商业账户。它更适合收取 B2B 款项、自由职业收入和跨境转账。

### 费率结构

| 费用类型 | 费率 | 说明 |
|----------|------|------|
| 接收本地转账 | 免费 | 使用 Wise 提供的本地银行账号 |
| 接收国际电汇 | 免费 | SWIFT 转账 |
| 货币转换 | 0.3%-1.5% | 视币种对而定，使用中间市场汇率 |
| 提现到银行 | 按币种不同 | 通常 $0-5 |
| 批量付款 | 每笔约 $0.5 | 适合给多个供应商付款 |

### 适用场景

- 自由职业者收取客户的银行转账
- 收取 AdSense、联盟营销佣金
- 向海外供应商和作者付款
- 作为 Stripe/PayPal 的提现中转账户

### 注册与使用

1. 访问 wise.com/business 注册商业账户
2. 提供营业执照或个人身份证件
3. 验证完成后获得多国本地银行账号
4. 将这些银行账号提供给客户进行转账
5. 收到的资金可在 Wise 内部兑换或提现

## 三大平台对比总结

| 维度 | Stripe | PayPal | Wise Business |
|------|--------|--------|---------------|
| **最适合** | 独立站在线收款 | 小额/个人交易 | B2B/自由职业 |
| **标准费率** | 2.9% + $0.30 | 3.49% + $0.49 | 0.3%-1.5%（换汇） |
| **技术门槛** | 中等 | 低 | 低 |
| **注册难度** | 高（需境外公司） | 低 | 中等 |
| **资金安全** | 高 | 中（冻结风险） | 高 |
| **到账速度** | 2-7 天 | 即时（PayPal 余额） | 1-3 天 |
| **多币种支持** | 135+ | 25 | 50+ |

## 实战建议

### 起步阶段（月收入 < $1000）

1. 先注册 PayPal 商业账户，用于接收小额付款和发送发票
2. 注册 Wise Business 账户，作为备用收款和换汇工具
3. 所有收入先以美元形式保留，等汇率合适时再转换

### 成长阶段（月收入 $1000-10000）

1. 注册香港或美国公司，开通 Stripe 账户
2. 网站主要收款渠道切换到 Stripe
3. 保留 PayPal 作为备用支付方式（部分用户偏好 PayPal）
4. 使用 Wise 处理非卡支付的收款

### 规模化阶段（月收入 > $10000）

1. 考虑使用 Stripe 的高级功能（Billing、Revenue Recognition）
2. 谈判定制费率（Stripe 对大客户提供优惠费率）
3. 考虑增加 LemonSqueezy 或 Paddle 处理全球税务合规
4. 设置自动化的财务报表和对账流程

## 安全与合规清单

- [ ] 开启所有平台的双因素认证（2FA）
- [ ] 设置交易通知，实时监控异常
- [ ] 定期检查 API 密钥和 Webhook 配置
- [ ] 保留所有交易记录至少 5 年
- [ ] 了解目标市场的税务要求（VAT/GST/Sales Tax）
- [ ] 制定退款和争议处理流程
- [ ] 分散风险，不要只依赖单一收款渠道
