# 拓词工具与方法

> 前几节我们从 Google 搜索、广告和竞品落地页中获取了种子关键词。但这些种子词还远远不够——你需要系统化地"拓词"，将几十个种子词扩展为成百上千的候选关键词。本节将介绍 SimilarWeb 关键词生成器和 Gemini AI 拓词两种高效方法。

---

## 一、SimilarWeb 关键词生成器

### 1.1 工具介绍

SimilarWeb 提供免费的关键词生成器（Keyword Generator），可以基于种子关键词快速生成大量相关词。与 Google Keyword Planner 不同，SimilarWeb 的数据来源于其独有的网络流量面板，能发现一些其他工具遗漏的关键词。

### 1.2 操作步骤

```
步骤 1：访问 SimilarWeb Keyword Generator
  │  https://www.similarweb.com/corp/marketing/keyword-generator-tool/
  │
  ▼
步骤 2：输入种子关键词（支持同时输入多个）
  │  例：project management, task management, team collaboration
  │
  ▼
步骤 3：选择目标国家和语言
  │
  ▼
步骤 4：查看生成结果，包含以下维度：
  │  ├── 关键词
  │  ├── 月搜索量
  │  ├── 搜索量变化趋势
  │  ├── CPC
  │  └── 竞争度
  │
  ▼
步骤 5：导出 CSV，进一步筛选和分析
```

### 1.3 SimilarWeb 独有的关键词维度

| 维度 | 说明 | SEO 应用 |
|------|------|---------|
| Search Volume Trend | 搜索量变化趋势曲线 | 判断关键词是上升还是下降 |
| Organic vs Paid | 自然搜索和付费搜索的流量占比 | 高有机占比的词更适合 SEO |
| Traffic Share | 关键词带给各网站的流量份额 | 了解谁在这个词上获得最多流量 |
| Zero-click Rate | 零点击率（用户不点击任何结果） | 零点击率高的词 SEO 价值有限 |

### 1.4 高效使用技巧

**技巧一：竞品关键词反查**

在 SimilarWeb 中输入竞品域名，查看其流量来源关键词：

```
SimilarWeb → Website Analysis → 输入竞品域名
→ Search → Organic Keywords
→ 查看竞品获取有机流量的关键词列表
```

**技巧二：关键词交叉分析**

对比多个竞品的关键词来源，找到共同流量词和差异化机会：

```
竞品 A 的关键词  ∩  竞品 B 的关键词  =  行业通用高价值词
竞品 A 的关键词  -  竞品 B 的关键词  =  差异化机会词
```

**技巧三：关注 "Trending Keywords"**

SimilarWeb 会标注搜索量快速增长的关键词（Trending），这些是尚未完全竞争的增长型关键词。

## 二、Gemini AI 拓词

### 2.1 为什么用 AI 拓词

传统关键词工具依赖搜索引擎历史数据，而大语言模型（LLM）能够理解语义关联、模拟用户思维、生成创意组合。两者互补使用效果最佳：

| 维度 | 传统关键词工具 | AI 大模型 |
|------|--------------|----------|
| 数据来源 | 搜索引擎历史数据 | 训练语料中的语义知识 |
| 搜索量数据 | 精确提供 | 无法提供（需交叉验证） |
| 语义理解 | 基于字面匹配 | 深度语义理解 |
| 创意发散 | 基于已有查询 | 能生成全新组合 |
| 用户意图理解 | 基于分类模型 | 能模拟真实用户思维 |

> **核心原则**：AI 生成的关键词必须通过搜索量数据验证。AI 负责"发散"，工具负责"验证"。

### 2.2 Gemini 拓词 Prompt 模板

**模板一：种子关键词拓展**

```
Prompt：

我正在做 [行业/产品] 的海外 SEO 关键词研究。
种子关键词是：[关键词列表]

请帮我完成以下任务：
1. 为每个种子关键词生成 10 个语义相关的关键词变体
2. 包含同义词、近义词、上下位词
3. 涵盖不同搜索意图（信息型、商业调查、交易型）
4. 考虑美国/英国英语的表达差异

输出格式：表格形式，包含关键词、搜索意图分类、与种子词的关系
```

**示例输出**：

| 种子词 | 拓展关键词 | 搜索意图 | 关系类型 |
|--------|-----------|---------|---------|
| CRM software | customer relationship management tool | 信息型 | 同义词 |
| CRM software | best CRM for small business | 商业调查 | 长尾扩展 |
| CRM software | CRM vs spreadsheet | 商业调查 | 对比词 |
| CRM software | free CRM download | 交易型 | 修饰词 |
| CRM software | sales pipeline management | 信息型 | 功能词 |

**模板二：用户画像驱动拓词**

```
Prompt：

我的目标用户画像是：
- 角色：[职位/身份]
- 公司规模：[大小]
- 痛点：[主要挑战]
- 购买阶段：[意识/考虑/决策]

请模拟这个用户在 Google 中可能搜索的关键词，包括：
1. 在意识阶段会搜索什么（不知道解决方案）
2. 在考虑阶段会搜索什么（对比不同方案）
3. 在决策阶段会搜索什么（准备购买/注册）

每个阶段列出 10 个可能的搜索查询。
```

**模板三：角色扮演法拓词**

```
Prompt：

请分别扮演以下角色，告诉我他们在 Google 搜索 [主题] 相关信息时
会使用什么关键词：

1. 一个完全不懂技术的中小企业老板
2. 一个正在做市场调研的产品经理
3. 一个想省钱的创业者
4. 一个需要向老板汇报的市场总监
5. 一个正在写评测文章的博主

每个角色列出 5 个最可能的搜索查询。
```

**模板四：竞品对比关键词生成**

```
Prompt：

我的产品是 [产品名]，主要竞品包括 [竞品A]、[竞品B]、[竞品C]。

请生成以下类型的关键词：
1. "[我的产品] vs [竞品]" 对比关键词（所有竞品组合）
2. "[竞品] alternative" 替代方案关键词
3. "[竞品] pros and cons" 优缺点关键词
4. "switch from [竞品] to [我的产品]" 迁移关键词
5. 功能对比关键词（如 "[产品A] [功能] vs [产品B] [功能]"）
```

### 2.3 结构化输出提高效率

要求 AI 以 CSV 格式输出，方便直接导入 Google Sheets 或关键词工具：

```
Prompt 追加指令：

请以 CSV 格式输出结果，列包括：
keyword, search_intent, relevance_score, suggested_content_type, notes

其中：
- search_intent: I(信息型)/C(商业调查)/T(交易型)/N(导航型)
- relevance_score: 1-5 分
- suggested_content_type: blog/landing/comparison/faq
```

## 三、AI + 工具的组合拓词工作流

单独使用 AI 或单独使用工具都不够。最高效的拓词方式是两者结合：

```
阶段 1：AI 创意发散
  │  使用 Prompt 模板生成 100-200 个候选关键词
  │
  ▼
阶段 2：去重和清洗
  │  去除重复、不相关、不通顺的关键词
  │  预期保留 50-100 个
  │
  ▼
阶段 3：数据验证（SimilarWeb / Keyword Planner）
  │  批量查询搜索量和 KD
  │  排除搜索量为 0 或极低的词
  │
  ▼
阶段 4：AI 意图分类
  │  让 AI 批量分类搜索意图
  │  标注关键词与业务的关联度
  │
  ▼
阶段 5：优先级排序
  │  按搜索量 x 相关度 / KD 排序
  │  确定 Quick Wins（快速见效词）和战略关键词
  │
  ▼
阶段 6：内容规划
     使用 AI 为优先关键词生成内容大纲
```

## 四、AI 拓词的注意事项

### 4.1 必须验证

AI 生成的关键词**不能直接使用**，必须通过搜索量数据验证：

| AI 可以做的 | AI 不能做的 |
|------------|-----------|
| 语义拓展和关联 | 提供准确搜索量 |
| 模拟用户搜索行为 | 判断关键词难度 |
| 生成创意组合 | 保证关键词有搜索量 |
| 批量分类搜索意图 | 替代专业 SEO 工具 |

### 4.2 提高 AI 拓词质量的技巧

- **提供上下文**：告诉 AI 你的行业、目标市场、用户画像
- **分批处理**：每次处理 20-30 个关键词，避免上下文过长导致质量下降
- **迭代优化**：根据验证结果反馈给 AI，调整 Prompt 后重新生成
- **多模型交叉**：用 Gemini、ChatGPT、Claude 分别生成，取并集
- **记录好 Prompt**：将效果好的 Prompt 模板保存为团队资产

### 4.3 AI 的局限性

- **无法提供搜索量数据**：如果你要求 AI 提供搜索量，它可能会编造——不要信任
- **知识有截止日期**：最新趋势和新兴关键词可能无法覆盖
- **语言偏差**：对于小语种市场，AI 的关键词建议可能不够地道

## 五、实战案例：从 5 个种子词拓展到 200+ 关键词

```
种子词（5 个）：
email marketing, newsletter tool, email automation,
email campaign, email list building

拓词过程：

1. Google 搜索下拉词 → 获得 60 个关键词
2. Keyword Planner 拓展 → 获得 120 个关键词
3. SimilarWeb 补充 → 获得 40 个关键词
4. Gemini AI 语义拓展 → 获得 80 个关键词
                         ─────────────
                         合计 300 个（去重前）

5. 去重 → 剩余 210 个
6. 搜索量验证（排除搜索量=0）→ 剩余 165 个
7. 业务相关度筛选 → 最终 120 个有效关键词

关键词分布：
├── 核心词（Head）：8 个
├── 中腰词（Body）：35 个
└── 长尾词（Tail）：77 个
```
